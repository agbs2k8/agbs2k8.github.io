Technology Skills

There is always a new programming language, database technology or some new framework to learn. Below is a collection of some of the more important technologies that I commonly work with, use on particular projects, or at least feel like I could dust-off and use again if needed.

Coding

I work primarily in Python (Anaconda), but I'm also familiar, though out of practice, with Javascript, R, Scala, Java and Apex

Databases

I work with SQL (MS/tSQL, SQLite, and PostgreSQL) more than anything, but have also done projects using MongoDB and the Neo4j graph database.

Frameworks

I know and regularyl work with SciKit-Learn, Keras/Tensorflow, Numpy, Pandas, SQLAlchemy, and Requests.

Algorithms

Every data scientist has their favorite, go-to algorithms. These are my favorites: Deep Learning (including Convolutional and Recurrent Neural Networks), Linear Regression, Logistic Regression, Support Vector Machines, Decision Trees, Bayes, Time Series (SARIMA)

Data Visualization

As a lover of python, I of course default to using Matplotlib, Seaborn, and Plotly. I'm familiar with ggplot in R, and next up to learn is D3.js.

Fun Projects

If you have never played with a Raspberry Pi, then you are missing out! They are tons of fun and I have built Hadoop Clusters, IOT devices, image-recognition tools and more with them.
